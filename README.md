# 📊 Business Insights 360 – AtliQ Hardware

## 🚨 Problem Statement
AtliQ Hardware, like many growing businesses, faced significant losses in the **Latin America region** due to limitations in Excel-based analytics.

To overcome this challenge, senior executives initiated a **data analytics transformation project**, leveraging historical data to improve performance, uncover insights, and enable better decision-making.

---

## 📊 Power BI Techniques Applied

- Created **Calculated Columns & Measures** using DAX  
- Built **robust data models** with relationships  
- Implemented **Bookmarks** for dynamic visual switching  
- Designed **Page Navigation using buttons**  
- Used **DIVIDE()** to handle zero division errors  
- Created **Date Table using M Language**  
- Developed **Dynamic Titles** based on filters  
- Applied **KPI Indicators** for performance tracking  
- Used **Conditional Formatting** (icons & colors)  
- Performed **Data Validation techniques**  
- Published reports to **Power BI Service**  
- Configured **Personal Gateway for Auto Refresh**  
- Managed **Workspaces, Access & Collaboration**
